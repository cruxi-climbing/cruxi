

# Developing

### Requirements :
- [Bun](https://bun.com/)
- Docker [[Docker Desktop](https://www.docker.com/products/docker-desktop/) | [Podman desktop](https://podman-desktop.io/) | [Orbstack](https://orbstack.dev/)]
- [BiomeJS](https://biomejs.dev/fr/)
- [iOS simulator](https://developer.apple.com/xcode/) | [Android studio simulator](https://developer.android.com/studio?hl=fr)


### Steps :

#### 1. Install dependencies
```bash
bun install
```

#### 2. Setup
```bash
bun setup
```

#### 3. Start dev servers

```bash
bun dev
```


#### 4. Start mobile app
In the interactive expo dev server :
- Press `i` to start iOS simulator
- Press `a`to start Android simulator
